# Daily Habits - 1% Better Every Day

_Inspired by James Clear's Atomic Habits. The goal is one intentional improvement per day in each area._

## Pillars

1. **Pillar 1** - (Your primary project or skill - e.g., "Second Brain", "Side Project", "Learning Go")
2. **Pillar 2** - (Your community or professional growth - e.g., "Open Source", "Team Leadership", "Content Creation")
3. **Pillar 3** - (Your most important relationship - e.g., "Marriage", "Family", "Friendship")
4. **Pillar 4** - (Your spiritual or mental health - e.g., "Meditation", "Journaling", "Faith")
5. **Pillar 5** - (Another project or area of growth - e.g., "Fitness", "Reading", "Side Business")

_Customize these pillars to reflect the 3-5 areas of your life where consistent daily effort compounds into something meaningful. Remove or add pillars as needed._

## Auto-Detection Rules

The heartbeat can auto-check a pillar ONLY when these criteria are met.
Routine maintenance does NOT count - it must be a genuine improvement.

**Pillar 1:**
- Auto-check when: A significant new feature is shipped, a meaningful architectural improvement is made
- Does NOT count: Bug fixes, config changes, routine maintenance

**Pillar 2:**
- Auto-check when: New content published, event hosted, major community contribution
- Does NOT count: Routine replies, minor updates

**Pillar 3:**
- NEVER auto-check. You must report this yourself.

**Pillar 4:**
- NEVER auto-check. You must report this yourself.

**Pillar 5:**
- Auto-check when: Meaningful commit, new feature shipped, measurable progress
- Does NOT count: Dependency updates, trivial changes

_Adjust these rules to match your pillars. The key principle: auto-detection is for objectively verifiable improvements. Personal and relational pillars should always be self-reported._
